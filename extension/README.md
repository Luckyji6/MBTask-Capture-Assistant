# ManageBac 菜单助手扩展

一个 Manifest V3 Chrome 扩展，用于在 ManageBac 页面内抓取 `a.f-menu_link.f-menu_submenu-link` 菜单链接，并在扩展弹窗里查看。

## 安装（开发模式）

1. 打开 `chrome://extensions/`，开启右上角“开发者模式”。
2. 点击“加载已解压的扩展程序”，选择本目录 `extension/`。
3. 浏览器工具栏会出现“ManageBac 菜单助手”图标。

## 使用步骤

1. 在新标签页访问 `https://{school}.managebac.cn/` 并登录。
2. 登录后点击浏览器工具栏中的扩展图标：
   - 弹窗会显示最近一次缓存的菜单链接。
   - 点击“刷新当前标签”将向当前 ManageBac 标签注入抓取指令，并更新列表。
3. 如提示“未找到菜单链接”，通常是因为尚未进入包含左侧菜单的页面或页面尚未完全加载。

## 工作原理

- `content.js` 在 `https://*.managebac.cn/*` 页面注入，监控 DOM 是否出现目标菜单，并向后台发送抓取结果。
- `service_worker.js` 缓存最近一次抓取的数据，并响应弹窗的读取与刷新请求。
- `popup.html/js` 读取缓存、显示菜单，可触发当前标签重新抓取或自动打开第一个班级的任务页面，并展示最近一次任务数量统计。

## 任务统计

- 弹窗中会显示最近一次在 `/core_tasks` 页面抓取到的任务卡片数量，并列出任务名称、成绩/分值、截止日期以及各个评估标准（如 A/B/C 等）的得分，同时标注提交状态（未交/已交/逾期）。
- 点击“获取任务与成绩”后，扩展会导航到第一个班级的 `core_tasks` 页面，并在任务列表渲染完成后更新统计。
- 点击“抓取所有班级”后，扩展会依次打开侧边栏中的每个班级，收集各班级的任务汇总，并在弹窗底部显示进度与结果（请保持浏览器页面前台打开；若中途终止，可再次点击按钮重新开始）。
- 批量抓取完成后，可点击“导出 CSV”下载结果（包含班级名称、任务列表、截止时间、成绩/分值、评估细则以及提交状态）。

## 权限说明

- `host_permissions`: 仅限 `https://*.managebac.cn/*`，用于在学校域名内注入脚本。
- `tabs`: 用于弹窗获知当前活动标签并发送刷新请求。
- `storage`: 存储最近一次抓取结果，方便离开页面或关闭弹窗后仍可查看。
- `downloads`: 支持在弹窗中导出批量抓取的 CSV 文件。
- `scripting`（MV3 要求）: 由 Manifest 自动包含，用于与内容脚本通信。

## 常见问题

- **弹窗提示“请先在 ManageBac 页面里打开扩展再刷新”**  
  需要先切换到 ManageBac 标签页，再点击扩展按钮。

- **抓取不到菜单**  
  请确保已经登录并加载了左侧菜单，或者尝试点击页面内的菜单展开。

- **跨域/安全性**  
  扩展仅在用户本地浏览器运行，不会把数据发送到外部服务器。
